    Spielanleitung für 10 000:

  • Der jüngste Spieler beginnt.
  • Jeder Spieler kann pro Runde 3 mal würfeln, wobei er nach jedem Wurf entscheiden kann, welche Würfelergebnisse er stehen läßt.
  • Würfelt er z. B. beim ersten Mal 1,3,5,5,6 so kann der Spieler die beiden Fünfen stehen lassen und mit den 3 restlichen Würfeln noch einmal würfeln. Spätestens nach dem 3. Wurf werden ihm die Punkte, die er erzielt hat, gutgeschrieben.

    • Spielende

Punkte:

5er = 50 Punkte

1er = 100 Punkte

    3er Päsche:
  • 3 x 1 = 1.000 Punkte
  • 3 x 2 = 200 Punkte
  • 3 x 3 = 300 Punkte
  • 3 x 4 = 400 Punkte
  • 3 x 5 = 500 Punkte
  • 3 x 6 = 600 Punkte

Straße (1-5 oder 2-6) = 2.000 Punkte
